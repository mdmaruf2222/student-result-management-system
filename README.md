Student Result Management System

A simple web-based application built with PHP and MySQL to manage student academic results. The system allows administrators to add, view, and delete student records with full CRUD functionality and database integration.

Features

Add student results

View and delete records

Admin-style management

MySQL database integration

Technologies

PHP

MySQL

HTML

CSS

XAMPP

Setup

Install XAMPP and start Apache & MySQL

Place project folder in htdocs

Import database.sql into phpMyAdmin

Open http://localhost/student_result_system/
